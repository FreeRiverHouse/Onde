
## Formato File

Ogni file giornaliero contiene:
- Idee e decisioni del giorno
- Frasi chiave di Mattia
- Riferimenti e link utili

## Workflow Automatico

Il workflow `.github/workflows/daily-ideas-digest.yml` manda ogni giorno alle 20:00 PT un riepilogo su Telegram con:
- Le idee del giorno
- Numero di commit nelle ultime 24h
