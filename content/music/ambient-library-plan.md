## Workflow Produzione

### Fase 1: Setup (prima della generazione)
1. Creare cartella output: `/Volumes/DATI-SSD/onde-music/ambient-library/`
2. Sottocartelle: `lullabies/`, `study/`, `relax/`, `nature/`
3. Preparare spreadsheet tracking

### Fase 2: Generazione (con Suno)
1. Usare Suno web interface (non API - costa)
2. Generare 5-10 tracce per sessione
3. Selezionare le migliori varianti
4. Download immediato in cartella corretta

### Fase 3: Post-Produzione
1. Normalizzare a -14 LUFS
2. Verificare qualita (no artefatti, no clipping)
3. Aggiungere fade in/out (2-3 sec)
4. Rinominare secondo naming convention
5. Aggiungere metadata ID3

### Fase 4: Archiviazione
1. Backup su SSD esterno
2. Upload su storage cloud (TBD)
3. Catalogazione per distribuzione

- **NON usare Suno API** - richiede crediti a pagamento
