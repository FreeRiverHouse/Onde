## 2. Strategia di Posting

### Frequenza Consigliata

```

├── Martedì:   Arte Visiva

### Volume

- **Ottimale**: 2-3 post/giorno (distribuiti: mattina, pomeriggio, sera)
- **Massimo**: 5 post/giorno (evitare saturazione)

### Orari Migliori (IT timezone)
- 08:00-09:00 (pendolari)
- 12:30-14:00 (pausa pranzo)
- 18:00-20:00 (dopo lavoro)
- 22:00-23:00 (browsing serale)

## 3. Adattamento Formato IG → X

### Differenze Chiave

| Aspetto | Instagram | X (Twitter) |
|---------|-----------|-------------|
| Focus | Visual-first | Text-first con visual |
| Caption | Lunga, descrittiva | Breve, impattante |
| Hashtag | Molti (fino a 30) | **ZERO** |
| Engagement | Like, commenti | Repost, quote, reply |
| Algoritmo | Follower + Explore | For You + Following |

### Regole di Conversione

1. **Immagini**
   - Verificare aspect ratio (X preferisce 16:9 o 1:1)
   - Comprimere se necessario (max 5MB per immagine)
   - Carousel IG → Thread X oppure singola immagine migliore

2. **Video**
   - Max 2:20 per video standard
   - Considerare ritaglio momenti salienti
   - Aggiungere sottotitoli se parlato

3. **Testo**
   - Da caption lunga → tweet conciso
   - Estrarre la frase più potente
   - Il resto può diventare thread se merita

## 4. Template Caption per Categoria

### POESIA

```
Template A (Verso Singolo):
[verso poetico]

Template B (Con Contesto):
[titolo se esiste]

[verso/frammento]

[data originale se significativa]

Template C (Riflessione):
[verso]

scritto in [contesto: notte insonne, viaggio, etc.]
```

**Esempio**:
```
il silenzio ha una texture
che solo i pixel conoscono

2019, somewhere in VR
```

### ARTE VISIVA

```
Template A (Minimal):
[titolo opera]
[anno]

Template B (Processo):
[titolo]

[una frase sul processo/ispirazione]

Template C (Serie):
[nome serie] — [numero/titolo]
[breve contesto se prima della serie]
```

**Esempio**:
```
Entropy Garden #7

quando il codice decide
dove fiorisce il caos
```

### VR / IMMERSIVE

```
Template A (Esperienza):
[nome esperienza/luogo VR]

[sensazione o osservazione]

Template B (Behind the Scenes):
building [cosa] in [piattaforma]

[insight tecnico o emotivo]

Template C (Invito):
[descrizione scena]

questo esiste in VR.
[dove trovarlo se pubblico]
```

**Esempio**:
```
walking through data streams
in a world I built from loneliness

VRChat, 3am, 2021
```

### TECH / CODE

```
Template A (Osservazione):
[osservazione su tech/AI/code]

Template B (Progetto):
[cosa hai costruito]
[perché/come in una riga]

Template C (Domanda):

**Esempio**:
```
ogni algoritmo è un'autobiografia

### FILOSOFIA

```

### BEHIND THE SCENES

## 5. Policy Hashtag

### REGOLA ASSOLUTA: ZERO HASHTAG SU X

**Perché no hashtag su X:**
- Sembrano spam/desperate
- Interrompono il flusso del testo
- L'algoritmo di X non li premia come IG
- Riducono l'eleganza del post
- Profilo professionale/artistico ≠ hashtag

**Alternativa per discoverability:**
- Parole chiave naturali nel testo
- Engagement con community relevanti
- Quote tweet di account affini
- Partecipazione a conversazioni organiche

**Eccezione unica:**
- Eventi live con hashtag ufficiale (conferenze, etc.)
- Solo se strettamente necessario

## 7. Metriche di Successo

### KPI Primari
- Impression per post
- Engagement rate (like + repost + reply / impression)
- Follower growth rate
- Profile visits

### KPI Secondari
- Click su link (se presenti)
- Quote tweet (segnale di risonanza)
- Salvataggi/bookmark
- DM ricevuti (interesse profondo)

### Target Realistici (primi 3 mesi)
- Engagement rate: 2-5%
- Follower growth: +10-20%/mese
- Impression medie: baseline + 50%

## 8. Note Finali

### Do
- Mantieni la voce autentica
- Lascia respirare i post (no over-posting)
- Rispondi ai commenti genuini
- Remixa contenuti che performano bene
- Usa il formato nativo X (testo > link esterni)

### Don't
- Non crosspostare identico da IG
- Non usare emoji eccessivi
- Non chiedere engagement esplicitamente
- Non postare solo per postare
- Non ignorare le risposte

### Tono di Voce
Il tono @magmatic__ è:
- Contemplativo ma non pretenzioso
- Tecnico ma accessibile
- Poetico ma non ermetico
- Personale ma non confessionale
- Curioso, mai cinico
