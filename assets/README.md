## Come Usare (Tecnica Image-to-Image)

### Per generare nuove scene con personaggi esistenti:

1. **Carica l'immagine di riferimento su Grok** (drag & drop)
2. **Usa prompt SEMPLICI**:
   - "stesso personaggio, ma in uno studio radio"
   - "stesso personaggio, ma che legge in una stalla di notte con stelle"
   - "stesso personaggio, ma al parco con bambini"
3. **Fai SEMPRE upscale** dopo la generazione

### Regole

- **MAI** generare personaggi da zero con solo testo
- **SEMPRE** usare l'immagine di riferimento per mantenere coerenza
- **NO Pixar, NO 3D, NO CocoMelon**
- **NO guance rosse**

---

## Aggiungere Nuovi Asset

1. Genera l'asset con Grok usando lo stile corretto
2. Salva in `~/Downloads/`
3. Copia nella cartella appropriata (`characters/` o `styles/`)
4. Aggiorna questo README
5. Commit su GitHub
