## Company
- Founded: 2012
- HQ: San Jose, CA
- Focus: Single-stack SASE for enterprise and service providers

## Product Portfolio

### Versa SASE
Unified platform delivering:
- SD-WAN
- ZTNA (Zero Trust Network Access)
- SWG (Secure Web Gateway)
- CASB (Cloud Access Security Broker)
- FWaaS (Firewall as a Service)
- Advanced Threat Protection

### Versa Secure SD-WAN
- Application-aware routing
- Multi-transport (MPLS, broadband, LTE/5G)
- Zero-touch provisioning
- Centralized orchestration (Versa Director)

### Versa Titan
- Cloud-delivered SASE for SMB/mid-market
- Simplified deployment
- MSP-friendly

## Key Differentiators

### Single Stack Architecture (VOS)
- All functions on one OS (Versa Operating System)
- Single policy engine
- One data path for security + networking
- No service chaining overhead

### Carrier-Grade Platform
- Used by 100+ service providers
- Multi-tenant by design
- Scale to millions of users

### Flexible Deployment
- Cloud PoPs (Versa Cloud Gateway)
- On-premises (Versa appliances)
- Hybrid (customer choice)

## Common Use Cases

1. **MPLS to SD-WAN Migration**
2. **Secure Remote Work**
3. **Branch Consolidation**
4. **Multi-Cloud Connectivity**
5. **Managed SASE Service (for SPs)**
