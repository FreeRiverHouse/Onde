Prompting

The Context Sandwich

Always wrap your request with context before and constraints after. AI performs best when it knows the full picture.

[Context] I'm building a Telegram bot for content approval. [Request] Add a command that shows pending items. [Constraints] Use TypeScript. Keep it under 30 lines.
Effectiveness: High
via @builder

Debugging

The Fresh Start Signal

If you've been debugging the same bug for 5+ iterations, the approach is wrong. Reset.

"Let's start fresh. Forget the previous approach. Here's what I actually need: [clearer description]"
Success rate: 90%
via @swyx
Prompting

Show, Don't Tell

Instead of describing what you want, show an example. AI learns patterns faster than specifications.

// Instead of "make errors like X", show: // Good format: { error: "NOT_FOUND", code: 404 } // Now create error for: unauthorized
Accuracy: +40%
via @karpathy
Tools

Voice + AI = Speed

Dictation is 3-4x faster than typing. Use Wispr Flow or similar to speak your prompts.

[Speak] "Create a function that validates email addresses" [AI generates] [Speak] "Now add plus addressing"
Speed: 3x faster
via @FreeRiverHouse
