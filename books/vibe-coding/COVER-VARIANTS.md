# Code Surfing - Varianti Copertina

## Concept
Bambino (Sofia-style) + Robot (AIKO-style) che surfano insieme su un'onda digitale/codice.
Metafora visiva del "vibe coding" - cavalcare l'onda dell'AI.

## Stile
- European watercolor, Onde style
- Beatrix Potter meets tech
- NO Pixar, NO 3D
- Luce dorata calda
- Toni: blu oceano + dorato + accenti tech
