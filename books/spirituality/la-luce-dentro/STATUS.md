## Istruzioni Generazione Immagini

**QUANDO BROWSER DISPONIBILE:**

1. Connetti Claude for Chrome
2. Naviga su `https://x.com/i/grok`
3. Clicca "Create Images"
4. Per ogni prompt in `la-luce-dentro_prompts.txt`:
   - Copia prompt
   - Genera immagine
   - Fai UPSCALE
   - Scarica in `~/Downloads/`
5. Sposta in `images/` con nome corretto
6. Controllo QC anatomia

**IMPORTANTE - Coerenza Personaggio:**
- Emma DEVE avere sempre capelli rossi e lentiggini
- Nonno DEVE avere sempre capelli bianchi
- Usare prima immagine come reference per le successive
