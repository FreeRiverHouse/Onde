## Stile Illustrativo: Folk Art Luzzati

Ispirato a **Emanuele Luzzati** (1921-2007), maestro dell'illustrazione italiana:
- Colori vivaci e contrastanti (blu notte, giallo oro, rosso corallo, verde acqua)
- Forme geometriche stilizzate
- Atmosfera teatrale e fiabesca
- Bordi decorativi colorati
- Sfondo scuro con elementi luminosi

2. Salva immagini in `images/` con naming convention
3. Aggiorna questo README
4. Rigenera PDF con script `create-piccole-rime-luzzati.js`
