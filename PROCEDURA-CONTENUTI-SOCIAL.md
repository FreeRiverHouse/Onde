# PROCEDURA CONTENUTI SOCIAL

## OBIETTIVO
Creare HTML/PDF rilassante in stile ONDE con SOLO i post social per la settimana. Niente strategie, niente analisi, SOLO CONTENUTI.

## PROCESSO

### 0. VERIFICA OBBLIGATORIA (PRIMA E DOPO)

```bash
# ESEGUI SEMPRE prima di iniziare e dopo aver finito:
./scripts/verify-social-content.sh [file.html]

# Se errori → STOP! Non procedere!
```

### 1. Raccolta Post Social
- Estrarre SOLO i post da pubblicare
- **NON specificare piattaforma** - solo contenuto
- **⚠️ NESSUN HASHTAG MAI** - direttiva Elon Musk (ripetuto per l'ennesima volta)
- **⚠️ ZERO HASHTAG** - non usarli, non metterli, non esistono
- Testi in inglese (usare agente PR, NON Gianni Parola)

### 2. Creazione HTML/PDF Stile ONDE
- Design rilassante, turchese, celeste
- Colori freschi e luminosi
- Font eleganti (Playfair Display + Crimson)
- Atmosfera marittima sofisticata

### 3. Formato Post
```
[LIBRO] - [NUMERO]

[Testo post esatto]

[Visual description se necessario]
---
```

### 4. Struttura Finale
- Copertina: "CONTENUTI SOCIAL - ON THE RELAX"
- Post raggruppati per **LIBRO** (es. Meditations 1,2,3... poi Psalm 23 1,2,3...)
- **NON specificare piattaforma** - solo contenuto
- 1 post per pagina/slida
- Footer: "@ONDE_FRH"
- Niente appendici, niente note

### 5. ⚠️ CHECK DUPLICATI (OBBLIGATORIO)
**PRIMA di finalizzare la lista, verificare:**

1. **Citazioni duplicate**: Nessun post deve usare la stessa citazione
2. **Confronto testo**: Se due post hanno la stessa frase principale → ELIMINARE quello con meno testo
3. **Verifica incrociata**: Controllare TUTTI i post contro TUTTI gli altri
4. **Regola**: Ogni citazione deve essere UNICA nella lista finale

**METODO:**
- Estrarre tutte le citazioni principali
- Cercare duplicati
- Se trovati: mantenere solo il post con più contesto/testo
- Ricontrollare dopo eliminazione

## OUTPUT RICHIESTO
HTML/PDF con SOLO i post social, stile ONDE rilassante, pronto per essere usato.
