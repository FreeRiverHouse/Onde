## PIPELINE PRODUZIONE NOTTURNA

### Workflow Automatico

```
1. QUEUE (sera)
   └── Aggiungi contenuto alla coda
       ├── Testo/script
       ├── Immagini base
       └── Parametri (lingua, voce, formato)

2. BATCH PROCESSING (notte, 2:00-6:00 AM)
   ├── ElevenLabs: Genera audio
   ├── Wav2Lip: Sync labiale (se video talking head)
   ├── FFmpeg: Assembla video
   └── Compress: Ottimizza per piattaforma

3. OUTPUT (mattina)
   ├── /output/youtube/
   ├── /output/shorts/
   ├── /output/podcast/
   └── /output/social/

4. REVIEW (Approval Dashboard)
   └── Mattia approva/rifiuta via Telegram
```
